## Що зроблено в цій серії сесій

1. **Вкладка «🎯 Цілі & план»** — перероблена з 3 під-вкладок на єдину ієрархію:
   **Цілі (🎯 фіолетовий #a855f7) → Плани дій (📋 cyan #06b6d4) → Задачі (✅ зелений #00ff88)**.
   - Розгортання/згортання рядків (клік по всьому рядку), стан у localStorage `ai_tracker_gp_exp`.
   - Soft-delete (× ставить `deletedAt`), секція «✗ Видалені» з відновленням (тримає макс 10).
   - Секція «✓ Досягнуто» внизу.
   - Інлайн-додавання дочірніх (план у ціль, задача в план).
   - XP за замовчуванням: задача=50, план=150, ціль=500.

2. **Напрями (streams)** — фільтр-теги на елементах: **📚 Розвиток (#6366f1), 🎬 Контент (#ec4899), 🚀 Проекти (#f59e0b)**.
   - Фільтр-бар зверху вкладки. Клік по тегу/`＋напрям` циклічно змінює напрям.
   - Нові елементи успадковують активний фільтр; дочірні — напрям батька.

3. **Прогрес-бар у цілях** — показує виконані/всього планів + %.

4. **Дашборд «Фокус»** — синхронізований з кольорами ієрархії, виключає soft-deleted.

5. **Кнопки 📌 (pin)** — непозначено = opacity 15%, позначено = золото + glow. Усі 6 кнопок (вкладка + Фокус).

## ⏳ ГОЛОВНЕ НЕЗАВЕРШЕНЕ ЗАВДАННЯ: «💡 Інбокс ідей»

Обговорено детально, **код ще НЕ написаний**. Суть:

- Окрема легка секція/вкладка «💡 Інбокс» — буфер для мозкового штурму (сирі ідеї одним рядком).
- **Масове вставлення**: користувач копіює список ідей (кілька рядків) → вставляє одним блоком → застосунок розбиває на окремі картки (по рядку = картка).
- Кожну картку можна **конвертувати одним кліком у ціль / план / задачу** (з вибором напряму) АБО видалити.
- Інбокс НЕ дає XP і не рахується в прогресі — це лише «передпокій» перед системою.
- Зберігати в localStorage (додати ключ у головний state-об'єкт, напр. `ideasInbox: [{id, text, createdAt}]`).

**Юзкейс:** користувач читає IndieHackers-розсилку → кидає текст у чат зі мною → я роблю вижимку ідей «без води» → він копіює список → вставляє в Інбокс → пізніше сортує в цілі/плани/задачі.

## Структура даних localStorage (ключ `ai_tracker_v1`)

Головний state-об'єкт зберігається в useEffect (шукай `const state = {`).
Релевантні масиви для вкладки:
- `longGoals` — Цілі: `{id, text, period, customXP, done, pinned?, stream?, deletedAt?, completedAt?, xpAwarded?, createdAt}`
- `plan` — Плани: `{id, text, type, urgency, xp, done, goalId?, pinned?, stream?, deletedAt?, ...}`
- `goals` — Задачі: `{id, text, priority, xp, done, planId?, pinned?, stream?, deletedAt?, ...}`

Вкладка рендериться як IIFE: `{activeTab === "goalsplan" && (() => { ... })()}`.
