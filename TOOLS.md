## Крок 1 — n8n локально (наступний крок 🔜)

**Що:** платформа візуальної автоматизації (як конструктор пайплайнів).
**Навіщо:** автоматизація рутини — контент, Telegram-бот, обробка даних.
**Коли:** зараз, як тільки буде час на «поекспериментувати».
**Ціна:** безкоштовно (Community Edition, self-hosted).
**Обмеження:** працює тільки поки ПК увімкнений і не в режимі сну.

### Встановлення (2 команди через Docker)
```bash
docker volume create n8n_data
docker run -it --rm --name n8n -p 5678:5678 -v n8n_data:/home/node/.n8n docker.n8n.io/n8nio/n8n
```
Потім відкрити: `http://localhost:5678`

### Перші автоматизації для тесту
1. **Telegram-бот «згенеруй пост»** — пишеш тему → отримуєш готовий пост
   - Telegram Bot API (безкоштовно назавжди)
   - Gemini API (безкоштовний tier, не треба платити)
2. **Репурпосинг контенту** — один текст → версії під кожну платформу
3. **AI-дайджест новин** — RSS → резюме → у Telegram

### Що треба мати
- Docker Desktop (безкоштовно)
- Telegram-бот через @BotFather (1 хв, безкоштовно)
- Gemini API ключ (безкоштовно: console.cloud.google.com)

## Крок 2 — n8n на VPS (коли n8n «зайшов» 📌)

**Що:** той самий n8n, але на хмарному сервері.
**Навіщо:** автоматизації 24/7 навіть коли ПК вимкнений/спить.
**Коли:** коли захочеш автопостинг за розкладом або будь-що «без тебе».
**Ціна:** ~$5/міс (Hetzner CX22 або DigitalOcean Droplet).

### Що відкривається після VPS
- Публічна адреса (напр. `n8n.твійдомен.com`)
- Цей чат (claude.ai/code) зможе **підключитись до n8n напряму** через MCP/API
- Claude читає і пише workflow живцем, бачить результати виконання
- Webhook-тригери (Instagram, форми, Stripe та ін.) починають працювати

### Провайдери
| Провайдер | Ціна | Примітка |
|---|---|---|
| Hetzner | ~€4/міс | найдешевший, сервери в Євро |
| DigitalOcean | $6/міс | зручний UI, є $200 кредит для нових |
| Contabo | ~€5/міс | більше RAM за ті ж гроші |

### Навички, які отримаєш заодно
- Основи Linux (must-have для будь-якого AI/tech проекту)
- Docker (стандарт індустрії)
- Домени + SSL (для будь-яких майбутніх сервісів)

## Крок 3 — MarkItDown MCP (коли підеш у роботу з документами 📌)

**Що:** MCP-сервер від Microsoft, конвертує файли → чистий Markdown.
**Навіщо:** надсилати PDF/Word/Excel в Claude без зайвих токенів.
**Коли:** коли почнеш регулярно працювати з PDF, Word, Excel, HTML.
**Ціна:** безкоштовно (open-source).

### Які формати підтримує
PDF, Word (.docx), Excel (.xlsx), PowerPoint (.pptx), HTML, зображення (OCR), ZIP

### Де реально зекономить
| Файл | Чому економія |
|---|---|
| PDF | сирець містить службові дані/координати/шрифти |
| Word/DOCX | XML-обгортка величезна, Markdown — чистий текст |
| HTML | теги/стилі/атрибути зникають, лишається зміст |

### Найсильніше в парі з n8n
```
PDF/Word → MarkItDown → Markdown → AI-вузол (дешевше ~70%) → результат
```
Приклади: PDF-бриф → резюме, Excel з товарами → описи, статті → пости.

### Джерело
`microsoft/markitdown` на GitHub. MCP-сервери є у кількох community-версіях.

## Instagram Comment → DM автоматизація

**Що:** людина пише ключове слово в коментарі → автоматичний DM з текстом/посиланням.
**Інструмент:** ManyChat (не n8n — у Instagram немає публічного API для коментарів).
**Безкоштовний план:** до 1000 контактів.
**Потрібно:** Instagram Business акаунт (безкоштовно).
**n8n роль:** опційно як розширення (ManyChat спіймав → n8n генерує текст через AI → відповідь).
